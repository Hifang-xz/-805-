\section*{二、简述，（共5题，35分）}
\begin{enumerate}
    \item 简述当入射在1/4波片上的光是自然光或者是偏振光时，从拨片出射的光的偏振性质如何？
    \item 如何用辩证的观点理解双缝的干涉与衍射？
    \item 两束光波可以产生干涉的条件是什么？如何使两个震动方向互相垂直的偏振光产生干涉？
    \item 根据你的理解，一般情况下，光波在各向同性介质和各向异性的介质中传播时有何不同？
    \item 什么叫光栅？其性质用何参数可以描述？它有什么作用和用途？
    \item 什么是瑞利判据？它对实际光学系统有何意义？
\end{enumerate}

\section*{三、计算与分析（共3题，20分）}

\begin{enumerate}
    \item 在迈克尔逊干涉仪实验中：测得空气层厚度变了0.25mm时，条纹变化了916个，求所用光源的波长？若空气层厚度变化时，从条纹出现到再次消失总共数得条纹3500条。\\（1）求光源发出的光波的谱线宽度为多少?\\(2) 将此光源用于玻璃薄板的干涉中，则薄板的极限厚度是多少？(\（n=1.5\）)

    \item \(\lambda=600nm\) 的单色光垂直入射到一光栅上测得第二级主级大衍射条角为30°，且第三级是缺级.\\(1)光栅常数d等于多少？\\(2)透光缝可能的最小宽度a等于多少？\\(3)在选定d和a后，求在屏幕上可能呈现的全部主级大的级次？
    \item 一列波长\(\lambda=1cm\) 的平面微波垂直入射在一个带有圆孔的不透明屏上，该孔的孔径是可调的。一个感光面积很小的探测器放在圆孔后中心轴上1m远处。如果圆孔的半径从零逐渐增大。问：\\(1)半径多大时，探测器能测到它的第一个最大值？\\(2)半径多大时，能在第一个最大值之后又测到第二个极小值？\\(3)在后一半径下，求沿轴的极大值和极小值位置所在？
\end{enumerate}
